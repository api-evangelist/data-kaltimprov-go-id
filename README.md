# Satu Data Kalimantan Timur (data-kaltimprov-go-id)

Satu Data Kalimantan Timur is a provincial government open-data portal for Indonesia running **CKAN**, exposing a standardized catalog API over ~12,388 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-kaltimprov-go-id/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-kaltimprov-go-id-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN

## API
- **Satu Data Kalimantan Timur CKAN Action API** — base `https://data.kaltimprov.go.id/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.kaltimprov.go.id)
- Live endpoint verified: `https://data.kaltimprov.go.id/api/3/action/package_search?rows=0` · ~12,388 datasets
